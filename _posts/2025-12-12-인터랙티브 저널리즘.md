<div class="step">
  <img src="/assets/images/inter-journalism/inter_intro.jpg" alt="솔루션 저널리즘 소개"/>
  <div class="text">
    <em><strong>인터랙티브 저널리즘이란?</strong></em>

    <span class="click-expand">
    인터랙티브 저널리즘은 독자가 단순히 뉴스를 ‘읽는’ 존재에 머무르지 않고,
    정보 탐색과 의미 구성 과정에 직접 개입하도록 설계된 저널리즘 입니다.
    </span>
    <span class="expand-note">
    즉, 정보의 ‘전달’이 아니라
    의미 형성의 과정에 독자를 참여시키는 방식입니다.
    </span>
    이는 클릭, 스크롤, 선택, 입력 등의 행위를 통해
    뉴스 소비 경험을
    능동적으로 전환시키는 데 목적이 있다고 볼 수 있습니다.

    이로써 단방향적 커뮤니케이션인 기존의 저널리즘을
    양방향적 커뮤니케이션으로 전환시켜 줍니다.
  </div>
</div>

<div class="step">
  <img src="/assets/images/inter-journalism/inter_process.jpg" alt="문제 확인"/>
  <div class="text">
    <em><strong>기술혁신과 인터랙티브 저널리즘</strong></em>

    인터랙티브 저널리즘의 등장은 데이터 처리 기술,
    <span class="click-word">웹 기반 시각화 도구, 사용자 인터페이스(UI)</span> 기술의 발전과
    밀접하게 연관되어 있습니다.

    대용량 데이터를 실시간으로 처리하고 시각적으로 구현할 수 있게 되면서
    복잡한 문제를 좀 더 직관적으로 이해하고
    재미있게 경험하게 해줍니다.

    또한 한편으로는 마음이 아프지만,
    지면 신문의 사양 산업화도 매우 큰 영향을 미쳤습니다.
  </div>
</div>

<div class="step">
  <img src="/assets/images/inter-journalism/inter_report.jpg" alt="결과 보고"/>
  <div class="text">
    <em><strong>레거시 미디어의 고착된 이미지 해결을 위하여</strong></em>

    개인적으로 저는 이러한 인터랙티브 저널리즘의 발달이
    현대 레거시 미디어의 한계를 뛰어넘을 방안으로
    기능할 수 있다고 생각합니다.
    <span class="click-focus">
    이는 분명 기성 언론의
    새로운 활로가 되어줄 수 있을 것입니다.
    </span>
    <span class="expand-note">
    다만 이 ‘활로’는 기술 그 자체가 아니라,
    기술을 통해 어떻게 이야기를 구성하고
    독자의 참여를 설계하느냐에 달려 있습니다.
    </span>
  </div>
</div>

<div class="step">
  <img src="/assets/images/inter-journalism/inter_examples.jpg" alt="실제 사례"/>
  <div class="text">
    <em><strong>실제 사례</strong></em>

    <span class="click-expand">
    바로 뉴욕타임즈의 Snow Fall 프로젝트입니다.
    </span>
    <span class="expand-note">
    해당 프로젝트는 2012년 12월에 공개되었고,
    워싱턴주 터널 크리크에서 발생한 스키어 눈사태 사고를 다루며
    큰 주목을 받았습니다.
    </span>
    <span class="sentence-toggle"
      data-alt="가령, 모바일 친화적이지는 않다는 점과 
      일정한 틀이 있는 것이 아니라서 추후에 
      다시 제작하려면 또 큰 비용과 시간을 소요한다는 점입니다.">
    해당 프로젝트는 사실 지금 공개된 타 프로젝트에 비해선
    기술적으로도 형식적으로도 아쉬운 점들이 눈에 보입니다.
    </span> 
    
    그러나 당시에 많은 언론사들이 차용하지 않던 방식을 선택해
    새로운 가능성을 실현한 프로젝트라는 점에서 인상깊었습니다.
  </div>
</div>

<div class="step">
  <img src="/assets/images/inter-journalism/inter_conclusion.jpg" alt="결론"/>
  <div class="text">
    인터랙티브 저널리즘은
    양방향적이고 혁신적인 저널리즘 사조라고 할 수 있습니다.
    <span class="focus-block">
    그러나 그만큼
    시간과 비용, 그리고 기획력이 요구된다는 점에서
    언론사에게 새로운 부담을 안겨주기도 합니다.
    </span>
    하지만 계속해서 새로운 형식이 등장하는 현재,
    <span class="final-question">
    언론사들은 "기존의 장점을 유지하면서도 어떻게 이를 독자에게 전달할 것인가"
    이러한 질문을 계속해서 던져야만 할 것입니다.
    </span>
    
  </div>
</div>

<script>
document.addEventListener("DOMContentLoaded", () => {

  document.querySelectorAll(".click-expand").forEach(el => {
    el.addEventListener("click", () => {
      el.nextElementSibling.classList.toggle("open");
    });
  });

  document.querySelectorAll(".click-focus").forEach(el => {
    el.addEventListener("click", () => {
      el.classList.toggle("active");
      el.nextElementSibling.classList.toggle("open");
    });
  });

  document.querySelectorAll(".click-word").forEach(word => {
    word.addEventListener("click", () => {
      word.classList.toggle("active");
    });
  });

  document.querySelectorAll(".sentence-toggle").forEach(el => {
    const original = el.innerHTML;
    const alternative = el.dataset.alt;
    let toggled = false;

    el.addEventListener("click", () => {
      el.classList.add("fading");

      setTimeout(() => {
        el.innerHTML = toggled ? original : alternative;
        el.classList.toggle("active", !toggled);
        toggled = !toggled;
        el.classList.remove("fading");
      }, 250);
    });
  });

});
</script>
