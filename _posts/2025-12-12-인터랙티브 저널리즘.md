<div class="step">
  <img src="/assets/images/inter-journalism/inter_intro.jpg" alt="솔루션 저널리즘 소개"/>
  <div class="text">
    <em><strong>인터랙티브 저널리즘이란?</strong></em>

    인터랙티브 저널리즘은 독자가 단순히 뉴스를 ‘읽는’ 존재에 머무르지 않고, 정보 탐색과 의미 구성 과정에 직접 개입하도록 설계된 저널리즘 형식이다. 
    
    이는 클릭, 스크롤, 선택, 입력 등의 행위를 통해 뉴스 소비 경험을 능동적으로 전환시키는 데 목적이 있다. 
    
    기사 내용은 하나의 고정된 서사라기보다, 독자의 선택에 따라 다른 맥락과 해석을 드러내는 구조를 띤다. 
    
    이러한 특성은 저널리즘을 일방향적 전달이 아닌 상호작용적 커뮤니케이션으로 재정의한다.
    
  </div>
</div>

<div class="step">
  <img src="/assets/images/inter-journalism/inter_process.jpg" alt="문제 확인"/>
  <div class="text">
    <em><strong>기술혁신과 인터랙티브 저널리즘</strong></em>

    인터랙티브 저널리즘의 등장은 데이터 처리 기술, 웹 기반 시각화 도구, 사용자 인터페이스(UI) 기술의 발전과 밀접하게 연관되어 있다. 
    
    대용량 데이터를 실시간으로 처리하고 시각적으로 구현할 수 있게 되면서, 복잡한 사회 문제를 단계적으로 탐색하는 보도가 가능해졌다. 
    
    또한 모바일 환경과 터치 기반 인터페이스는 독자의 참여를 자연스러운 행위로 만들었다. 기술은 단순한 도구를 넘어, 저널리즘 서사의 형식 자체를 변화시키는 조건이 되었다.

  </div>
</div>

<div class="step">
  <img src="/assets/images/inter-journalism/inter_research.jpg" alt="해결 사례 조사"/>
  <div class="text">
    <em><strong>기원과 초기 단체에 관해</strong></em>

    전통적인 레거시 미디어는 지면과 방송 포맷에 기반한 일방향 전달 구조라는 한계를 지닌다. 
    
    이 구조에서는 독자가 정보의 수용자에 머물며, 복잡한 이슈에 대한 이해 과정이 충분히 고려되지 않는다. 
    
    인터랙티브 저널리즘은 이러한 한계를 넘어, 독자가 자신의 속도와 관심에 맞게 정보를 탐색하도록 함으로써 이해의 깊이를 확장한다. 
    
    이는 뉴스에 대한 피로감과 무력감을 완화하고, 저널리즘의 효능감을 회복하려는 시도로 볼 수 있다.
    
  </div>
</div>

<div class="step">
  <img src="/assets/images/inter-journalism/inter_report.jpg" alt="결과 보고"/>
  <div class="text">
    <em><strong>레거시 미디어의 고착된 이미지 해결을 위하여</strong></em>

    대표적인 사례로는 미국 매체 The New York Times의 인터랙티브 그래픽 기사들을 들 수 있다. 
    
    들은 독자가 직접 데이터를 조작하거나 시나리오를 선택하면서 사회 현상을 체감하도록 설계되어 있다. 
    
    또한 The Pudding은 문화·사회 이슈를 인터랙티브한 스토리텔링으로 풀어내며, 분석과 체험을 결합한 저널리즘을 선보인다. 
    
    이러한 사례들은 정보 전달을 넘어 ‘이해의 경험’을 제공한다는 점에서 기존 기사와 차별화된다.
    
  </div>
</div>

<div class="step">
  <img src="/assets/images/inter-journalism/inter_examples.jpg" alt="실제 사례"/>
  <div class="text">
    <em><strong>실제 사례</strong></em>

    여러가지 사례가 있지만, 개인적으로 미국의 The Marshall Project를 소개해드리고자 합니다.

    The Marshall Project를 짧게 정의하자면, 미국의 형사 사법 제도 전문 비영리 언론 단체라고 할 수 있습니다.

    단순히 교도소를 조명하는 것이 아닌, 재범률을 낮추기 위한 목적과 이를 위해 각 주의 프로그램을 살펴보고

    이를 통해 왜 해당 프로그램을 그 주에선 적용했는지, 어떤 효과를 보였는지를 분석하여 보도합니다.

    제가 해당 프로젝트를 언급하는 이유는 법제 관련 분야에 관심이 있어서도 있지만

    동시에 해당 프로젝트는 단순히 솔루션 저널리즘이 성공을 위한 사례뿐만 아니라

    당해 프로그램은 <span class="underline">이러한 한계를 갖고 있으니 피해야한다는 점</span>을 잘 보여준다는 점에서 좋았던 것 같습니다.
    
  </div>
</div>

<div class="step">
  <img src="/assets/images/inter-journalism/inter_conclusion.jpg" alt="결론"/>
  <div class="text">
   솔루션 저널리즘은 분명, 단순 조명에 그치지 않고 우리가 나아갈 방향을 설정한다는 점에서

   더욱 적극적이고 생산적인 저널리즘 사조임에 틀림 없습니다.

   그러나 한편으로는 이를 위한 더욱더 긴 시간과 비용을 투자하는 취재라는 조건도 무시할 수는 없을 것입니다.
   
   <span class="focus-block">
  인터랙티브 저널리즘은 기술 발전의 산물이면서 동시에 저널리즘 위기에 대한 응답이다. 
     
  이는 형식적 혁신을 넘어, 독자를 저널리즘 과정의 일부로 다시 초대하려는 시도라는 점에서 의미를 가진다. 
  
  특히 레거시 미디어가 직면한 신뢰와 효능감의 위기를 완화할 수 있는 대안적 방향성을 제시한다. 결국 인터랙티브 저널리즘의 가치는 기술 그 자체가 아니라, 독자와 저널리즘의 관계를 재구성하는 데 있다.
   </span>
  </div>
</div>

document.addEventListener("DOMContentLoaded", () => {
  const targets = document.querySelectorAll(
    ".underline, .structure, .color-hold-soft, .focus-block"

  const observer = new IntersectionObserver(entries => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {

          entry.target.classList.add("active");
        } else {
          entry.target.classList.add("show");
        }
        observer.unobserve(entry.target);
      }
    });
  }, {
    threshold: 0.6
  });

  targets.forEach(el => observer.observe(el));
});
