  <div class="text">
    <em><strong>온라인 저널리즘을 수강하며</strong></em>

    이번 포스트는 온라인 저널리즘을 수강하며 느꼈던 점과 여러가지를 정리한 포스트입니다.
    
    개인적으로 정말 생각보다 너무나도 많은 서비스들이 존재한다는 걸 느꼈습니다.

    학우들의 발표를 들으며 알지 못했던 서비스들의 창업과정을 듣고,

    그 서비스들을 소개받으며 너무나도 즐거웠던 것 같습니다.

    그리고 창업이라는 것이 그렇게 거창한 것이 아님을 또 한편으로 느꼈던 것 같습니다.

    생각보다 가장 개인적인 문제는 공동체의 문제가 될 수 있었고,

    가장 나 다운 일도 생각하기에 따라 돈이 될 수 있다는 점이 흥미로웠습니다. 
    
  </div>
</div>

<div class="step">
  <img src="/assets/images/on-jour/process.jpg" alt="문제 확인"/>
  <div class="text">
    <em><strong>1. 에브리타임</strong></em>

    대학생이라면 누구나 아마 스마트폰에 깔려있을 어플리케이션인 

    에브리타임의 창업과정이 너무 재미있었습니다.
    
    내가 느끼는 불편함을 해결하기 위한 행동이 
    
    모든 대학생들의 고충 해결로 이어지기까지의 과정이 너무 재미있었습니다. 
    
    그렇기에 창업 아이템이라는 것이 그렇게 <strong style="color:#b8f8fb;">어려운 것이 아닐수도 있겠다</strong>고 생각을 바꿔볼 수 있었던 것 같습니다.
    
  </div>
</div>

<div class="step">
  <img src="/assets/images/on-jour/research.jpg" alt="해결 사례 조사"/>
  <div class="text">
    <em><strong>2. 옥천신문</strong></em>

    사실 옥천신문은 발표에서 다뤄지진 않았고, 교수님의 강의 중 언급되었습니다.

    전 지면신문이 사실 이제는 돌아올 수 없는 강을 건넌것이 아닐까...라고 만 생각하고 있었는데, 
    
    해당 사례를 보며 그럼에도 충분히 지면신문만이 가지는 장점이 있음을 다시 상기할 수 있었습니다.
    
    특히 소멸 위기에 처한 지역 언론이 나아가야 할 방향을 가장 잘 보여주는 사례라는 점에서 좋았던 것 같습니다.

    당근 마켓의 강의 내용도 개인적으로 그런 맥락에서 좋았지만,

    그래도 강의명이 저널리즘이니 만큼 <span class="underline">사회에 얼마나 공헌하는가</span>라는 점에서

    옥천신문이 이끌리는 느낌이었습니다.
    
  </div>
</div>

<div class="step">
  <img src="/assets/images/on-jour/report.jpg" alt="결과 보고"/>
  <div class="text">
    <em><strong>3. 더 푸딩</strong></em>

    더 푸딩은 이번에 코딩을 하면서 더 그 대단함을 느낄 수 있었습니다.

    소규모의 팀으로 그렇게 복잡한 데이터를 즐겁게 볼 수 있도록 설계하는 것이,

    그 코딩의 어려움도 어려움인데 어느 부분에 어떻게 효과를 넣을 것인가

    라는 점이 가장 대단하게 느껴졌던 것 같습니다.
    
    데이터 저널리즘이자 인터랙티브 저널리즘으로서의 그 가치를 정말 느낄 수 있었습니다.

    한국에도 저런 서비스들이 등장하길 기원합니다...!
    
  </div>
</div>

<div class="step">
  <img src="/assets/images/on-jour/examples.jpg" alt="실제 사례"/>
  <div class="text">
    <em><strong> 한 학기를 마치며 </strong></em>

    늘 매번 학기를 마치면, 하루 정도는 내가 이번학기를 통해 뭘 얻었나 생각을 해봅니다.

    이번 온라인 저널리즘 강의에서는 내가 뭘 느끼고 가져올 수 있었는지 내걸로 만들 수 있었나를 고민해보면,

    현재 내가 왜 커뮤니케이션을 배우고 있는가에 대한 대답을 얻을 수 있었던 것 같습니다.
    
    <span class="focus-block">
    지금 한국의 언론은 지역으로도 중앙으로도 큰 위기에 놓여있는 것 같습니다.

    이러한 상황 속 학부생으로서 배우는 이론과 문제들이 어떠한 의미를 가지는가를 
    
    사실 조금은 이해하지 못하고 있었던 것 같습니다.

    이에 이번 강의로 창업 과정과 서비스의 의의들을 들으며

    그 이론과 문제를 현실로 가져와 거창하진 않아도

    새로운 변혁을 만들어낼 수 있다는 그 사실을 알게된 것 같습니다.
    </span>
    
  </div>

<script>
document.addEventListener("DOMContentLoaded", () => {
  const targets = document.querySelectorAll(
    ".underline, .structure, .color-hold-soft, .focus-block"
  );

  const observer = new IntersectionObserver(entries => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        if (entry.target.classList.contains("underline")) {
          entry.target.classList.add("active");
        } else {
          entry.target.classList.add("show");
        }
        observer.unobserve(entry.target);
      }
    });
  }, {
    threshold: 0.6
  });

  targets.forEach(el => observer.observe(el));
});
</script>
